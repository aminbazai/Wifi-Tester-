# WiFi Security Demo Tool

A simple Python-based cybersecurity demonstration tool that explains the risks of weak WiFi passwords and how dictionary-based password guessing works.

This project is created for **educational purposes** to help students understand basic cybersecurity concepts and password security.

## Features

- Demonstrates the concept of password dictionary attacks

- Educational cybersecurity practice script
- Simple Python implementation
- Command line based tool
